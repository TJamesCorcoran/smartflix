[](arg,length=nil)

Brackets for easily accessing records by the numbers. Has three modes.

Pass a single ID and get a single record back. Example: 

  Model[1]  #=> #<Model id:1>

Pass in a range, and get all the records back with IDs in that range. Example: 

  Model[1..3]  #=> [#<Model id:1>,#<Model id:3>]

Pass in two integers, the first being the starting ID, and the second the length of the array of records to return. Example: 

  Model[1,2]  #=> [#<Model id:1>,#<Model id:3>]

first(arg={})

Find the first record for a particular Model, or find the first matching record by passing a hash of conditions.

Example: 

  Model.first #=> #<Model id:1>

Example: 

  Model.first(:foo => 'bar') #=> #<Model id:3>
